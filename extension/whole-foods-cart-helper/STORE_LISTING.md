# Chrome Web Store Listing Draft

## Name

Meal Helper Whole Foods Cart

## Summary

Review a Meal Helper shopping list and add saved Whole Foods products to an
Amazon cart.

## Single purpose

Transfer Whole Foods ingredients from the Meal Helper household meal plan into
a user-reviewed set of Amazon Whole Foods product mappings, then add those
mapped products to the cart without checking out.

## Permission justification

**Storage:** Saves preferred product mappings and the current cart plan in the
local Chrome profile.

**Host access to meals.andreicristea.com:** Detects a user-requested cart
transfer and receives only the locked Whole Foods ingredient list.

**Host access to amazon.com and wholefoodsmarket.com:** Lets the user select a
preferred product and allows a user-started cart run to locate and click the Add
to Cart control.

The extension does not use remotely hosted code, analytics, ads, or data sales.
